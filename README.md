# K.B. ASSOCIATES | Professional Loan Consultancy

A premium, high-performance financial services website built with Next.js 14 for **K.B. ASSOCIATES**, the leading loan consultancy in **Tiruppur**.

## ✨ Features

- **Premium Motion Design**: Smooth, staggered entrance animations and scroll-triggered reveals using `framer-motion`.
- **High-Impact Branding**: Enlarged, single-line typography for "K.B. ASSOCIATES" with localized sub-branding.
- **Modern Tech Stack**: Built with Next.js 14 App Router, TypeScript, and Vanilla CSS for maximum performance and polish.
- **Lead Capture**: Fully validated Contact Form with real-time feedback and Resend integration.
- **WhatsApp Integration**: Floating high-visibility chat widget for instant customer engagement.
- **SEO Optimized**: Advanced metadata configuration, structured data, and semantic HTML for local search dominance.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Forms**: [React Hook Form](https://react-hook-form.com/) + [Zod](https://zod.dev/)
- **Email**: [Resend](https://resend.com/)

## 🚀 Getting Started

### Prerequisites

- Node.js 18.17 or later
- [npm](https://www.npmjs.com/) or [pnpm](https://pnpm.io/)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-repo/kb-associates.git
   cd kb-associates
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Setup environment variables (`.env.local`):

   ```env
   RESEND_API_KEY=re_your_api_key
   CONTACT_EMAIL=kbassociates8587@gmail.com
   ```

4. Start development server:

   ```bash
   npm run dev
   ```

## 📈 Deployment

Recommended deployment on [Vercel](https://vercel.com/):

